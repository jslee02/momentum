Detailed Description

template<typename T>
struct momentum::DeduceSpanType< T >

A utility struct that facilitates the deduction of a gsl::span type from a given type.

This utility is particularly useful when a function accepts a gsl::span<Vector3<T>> as an argument, where the template argument of gsl::span is also a template. In such cases, direct deduction of the gsl::span type from other container types (e.g., std::vector<Vector3<T>>) at the call site may fail.

By using this utility, the appropriate gsl::span type can be deduced automatically, ensuring smoother function calls and enhancing code readability.

Example usage:

void foo(DeduceSpanType<Vector3f>::type points); // Equivalent to gsl::span<Vector3f>
std::vector<Vector3f> points;
foo(points); // This line will compile without any issues.
