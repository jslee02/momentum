Creates a random affine transform.

Parameters
[in]Trueto set translation to a random vector where each value is between -1 and 1. Otherwise, the translation component will be zero.
[in]Trueto set rotation to a unit random quaternion. Otherwise, the rotation component will be identity.
[in]Trueto set scale to a random value between 0.5 and 2.0. Otherwise, the scale component will be 1.
