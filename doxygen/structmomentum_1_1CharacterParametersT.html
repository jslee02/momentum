Detailed Description

template<typename T>
struct momentum::CharacterParametersT< T >

A struct that encapsulates both pose and identity parameters for a character.

Note
This structure implies, but does not enforce, that the pose vector should contain only pose information with identity-related elements set to zero. Similarly, the identity vector should exclusively contain bone length information, excluding any pose data.
