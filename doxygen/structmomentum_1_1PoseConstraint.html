std::vector<std::pair<size_t, float> > momentum::PoseConstraint::parameterIdValue

A vector of tuple of type: (model parameter index, parameter value).

The model parameter index must be in [0, numModelParameters.size()). The value of the model parameters specified here are kept constant (= parameter value) during optimization. The ordering of elements in the vector doesn't matter (making it an unordered map would be more semantically correct) since it stores an index to the model parameters
