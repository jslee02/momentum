Detailed Description

template<class T, std::size_t Alignment>
class momentum::AlignedAllocator< T, Alignment >

An allocator that aligns memory blocks according to a specified alignment.

The allocator is compatible with std::allocator and can be used in place of std::allocator in STL containers.

Template Parameters
TThe type of elements that the allocator will allocate.
AlignmentThe alignment for the allocated memory blocks.
